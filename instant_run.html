INSTANT RUN

About the project

A lightweight tool for launching applications and opening files from anywhere by simply searching for the name.


The project is available on Github.

There are also pre-built binaries.

Duration

since Jul 2025

Tech

C++

OpenGL

The project started out of frustration of using other solutions like Power Toys Run or just the Start menu search in Windows.

The former was getting too slow and unresposive especially when the power mode of my laptop was set to battery saver.

Whilst the latter was an okay option, the annoyance of having to deal with it launching Microsoft Edge the moment I misspelled the search query, outweighted all the benefits of it for me.


* Besides that, it was also an excuse for me to finally try writing my own UI from scratch, even if it is very minimalistic and limited ...


Although majority of the project was written from scratch, it still relies on 3 libraries: stb_truetype, stb_image and glad.

One of the main goals of the project was to make the searching be resposive, that is why it is done on every keystroke.

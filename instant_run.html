INSTANT RUN

About the project

A lightweight tool for launching applications and files from anywhere by simply searching for the name.


The project is available on Github.

There are also pre-built binaries.

Duration

since Jul 2025

Tech

C++

OpenGL

The majority of the project was written from scratch, however it still relies on 3 libraries: stb_truetype, stb_image and glad.

The app has a minimalistic UI, which was achieved using simple custom immediate mode GUI.

One of the main goals of the project was to make the searching be resposive, that is why it is being done on every keystroke.
